How to Use READ QURAN EASILY

Basic Navigation

  • Use the Surah dropdown menu to select any chapter of the Quran
  • Scroll through the verses to read the Arabic text and translation
  • Click the verse options menu (three dots) to access additional features

Text Customization

  • Access font size controls in Settings to adjust Arabic and Translation text size
  • Toggle translation visibility using the checkbox in Settings
  • Switch between light and dark themes for comfortable reading

Bookmarks

  • Click the bookmark icon in verse options to save verses
  • Access your bookmarked verses through the Bookmarks button
  • Click on any bookmarked verse to quickly navigate to it

Sharing and Export

  • Use the share option to share verses on social media
  • Take screenshots of verses for saving or sharing
  • Copy Arabic text or translation separately

Additional Features

  • View current Islamic date through the calendar icon
  • Access settings menu for additional customization options
  • Use the desktop version for the best experience
